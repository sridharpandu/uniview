airpay payment services PHP Kit - Readme
=========================================

Version 2.0.0

A set of PHP-scripts Examples to create payment solution for airpay payment services.

http://www.airpay.co.in/

Requirements
------------

* PHP 5.1 or later
* Merchant Account of airpay payment services.
* An Editor to Edit Code.
* a web-browser

Summary
-------

PHP Kit is intended to provide example to Merchant for integrating merchant Payment Solution.
For a summary of features, please see the Documentation.pdf file.
